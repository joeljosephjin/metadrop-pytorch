# metadrop-pytorch

Unofficial PyTorch Implementation of "Meta Dropout: Learning to Perturb Latent Features for Generalization" (ICLR 2020)

To Run:
`python main.py --phi`

## Results

|       | Omni. 1shot (main paper)| Omni. 1shot (Ours)|
| ------| ---------------- | ---------------- |
| MAML | 95.23          | 96.76 |
| Meta-dropout | __96.63__ | __97.12__ |
